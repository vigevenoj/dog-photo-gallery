I don't know about you, but I love seeing pictures of my dogs. I like being reminded about what they were up to a year ago, and I'll probably want to be reminded about their past antics on any given day of the year.

I couldn't find a photo-sharing site that really met my needs, so I wrote this.

It's a basic photo-displaying site with an API that suits my needs, storing photo metadata in PostgreSQL, image data in an s3-compatible object storage system, and using imgproxy to handle image resizing operations.


Source code is available at github.com/vigevenoj/dog-photo-gallery
